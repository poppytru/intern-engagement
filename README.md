# Amazon Intern Celebrations Webpage

A branded webpage for Amazon intern celebrations featuring upcoming events, quarterly prize engagements, and a tiered reward system.

## Features

- Upcoming celebration events across different hub cities
- Quarterly engagement challenges that unlock on specific dates
- Three-tier prize system with Amazon swag
- Fully responsive design
- Amazon brand guidelines compliant

## Brand Guidelines Applied

### Color Palette
- Squid Ink: #161D26 (primary text, header)
- Smile Orange: #FF6200 (accents, CTAs)
- White: #FFFFFF (cards, backgrounds)
- Off-White: #F5F3EF (page background)

### Typography
- Amazon Ember Display Bold for headlines
- Amazon Ember Display Regular for subheads
- Amazon Ember Regular for body copy
- Minimum 16px body text for accessibility

### Accessibility
- Proper color contrast ratios maintained
- Minimum text size of 16px (18px preferred)
- Semantic HTML structure
- Responsive design for all devices

## Setup

1. Replace placeholder images with actual event photos
2. Add Amazon logo file (amazon-logo.svg)
3. Open index.html in a web browser

## Notes

- Engagement cards automatically unlock on their scheduled dates
- The tiered prize system rewards based on engagement complexity
- Additional prizes available through quarterly recognitions
